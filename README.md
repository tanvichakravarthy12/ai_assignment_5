# AI Assignment 5

## Overview

This repository contains a collection of projects that demonstrate different AI concepts, including probabilistic reasoning, knowledge representation, intelligent planning, and game-playing algorithms.

The projects are implemented in Python.

---

## Projects

### 1. Bayesian Networks 

A probabilistic reasoning system that models uncertainty using Bayesian Networks.

**Topics Covered:**

* Bayesian Networks
* Conditional Probability Tables (CPTs)
* Probabilistic Inference
* Directed Acyclic Graphs (DAGs)

**Files:**

* `example.py`

---

### 2. Knowledge Graphs 

A simple implementation of Knowledge Graphs for representing and querying connected information.

**Topics Covered:**

* Knowledge Representation
* Semantic Triples
* Graph Traversal
* Network Analysis

**Files:**

* `graph.py`
* `network.py`

---

### 3. AI-Based Travel Planner

An intelligent travel planning system that generates personalized itineraries based on user preferences and budget.

**Topics Covered:**

* Rule-Based Recommendations
* Knowledge Bases
* Cost Optimization
* Personalized Planning

**Files:**

* `travel.py`
* `planner.py`
* `tourist_places.py`
* `food.py`
* `wine_ontology.py`
* `cost.py`

---

### 4. Adversarial Game Search Algorithms

A collection of AI search algorithms used for decision-making in competitive games.

**Algorithms Implemented:**

* Minimax
* Alpha-Beta Pruning
* Heuristic Alpha-Beta
* Monte Carlo Tree Search (MCTS)

**Games Supported:**

* Tic-Tac-Toe
* Connect Four

**Files:**

* `game.py`
* `code.py`
* `testcases.py`

---

## Technologies Used

* Python
* NetworkX
* Object-Oriented Programming
* Graph Theory
* AI Search Techniques
